# Serpent2 Microscopic Cross-Section Extraction Toolkit

## Overview

This repository contains the MATLAB workflow used to extract one-group microscopic cross sections and isotopic inventories from Serpent2 depletion calculations.

The extraction methodology processes Serpent2 depletion matrices (`depmtx`) and depletion results (`dep.m`) to generate:

* burnup-dependent microscopic cross sections;
* burnup-weighted average microscopic cross sections;
* burnup-dependent isotopic inventories.

The extracted data are intended for use within the Nuclear Material Balance (NMB) framework and can be employed in depletion and fuel cycle analyses.

This repository contains:

* MATLAB scripts implementing the extraction workflow;
* supporting MATLAB modules and nuclear data files;
* a minimal example Serpent2 case illustrating the required input structure;
* documentation required to reproduce the extraction procedure.

The complete Serpent2 benchmark dataset and the associated verification campaign are distributed separately through Zenodo.

## Input Data

### `data/`

The `data/` directory contains a minimal Serpent2 example case used to demonstrate the required input structure and to test the extraction workflow.

The complete Serpent2 dataset is not included in this GitHub repository because of its size and is available separately through Zenodo.

The example case is distributed as a compressed archive in order to reduce the repository size. Before running the extraction scripts, users should extract the contents of the archive and restore the original directory structure within the `data/` directory.

Once extracted, the example case can be used to verify the installation and test the complete cross-section extraction workflow without requiring the full benchmark dataset.

The extraction scripts assume the following directory hierarchy:

```text
data/
└── <library>/
    └── <REACTOR>/
        └── <enrichment>/
            ├── <reactor>
            ├── library
            ├── dep.m
            ├── depmtx*
            └── ...
```

where:

* `<library>` denotes the nuclear data library (e.g., ENDF/B-VIII.1, JEFF-4.0, JENDL-5.0);
* `<REACTOR>` denotes the reactor type;
* `<enrichment>` denotes the fuel enrichment case.

The main Serpent2 input file is named after the reactor type (`<reactor>`).

Each case includes a `library` file specifying:

* nuclear data library paths;
* thermal scattering law data.

Important Serpent2 output files used during post-processing are:

* `depmtx*`: burnup-dependent depletion matrices;
* `dep.m`: depletion calculation results.

To process additional Serpent2 calculations, users should place the corresponding output files within the same directory hierarchy.

## Output Data

### `results/`

The `results/` directory is automatically generated by the extraction workflow and is not included in this repository.

Depending on the executed script, the directory may contain:

* one-group microscopic cross sections formatted for NMB calculations;
* burnup-dependent microscopic cross sections;
* burnup-weighted average microscopic cross sections;
* burnup-dependent isotopic inventories extracted from Serpent2 depletion calculations.

The output hierarchy mirrors that of the input data:

```text
results/
└── <library>/
    └── <REACTOR>/
        └── <enrichment>/
```

A complete set of reference results generated from the full benchmark dataset is available through the associated Zenodo archive.

Generated files include:

### Library-Level Files

* `ALL_microXS_<library>.xlsx`

  Consolidated one-group microscopic cross sections for all reactor and fuel cases associated with the selected nuclear data library, formatted for NMB input.

### Case-Specific Files

For each reactor and enrichment combination:

* `<library>_<REACTOR>_<enrichment>_microXS.xlsx`

  One-group microscopic cross sections formatted for direct use in NMB calculations.

* `<library>_<REACTOR>_<enrichment>_microXS_byBU_FULL_and_ACTINIDES.mat`

  Burnup-dependent microscopic cross sections stored in MATLAB binary format.

* `<library>_<REACTOR>_<enrichment>_microXS_weightedAvg_ALL_isotopes_ORDERED.mat`

  Burnup-weighted average microscopic cross sections stored in MATLAB binary format.

* `<library>_<REACTOR>_<enrichment>_SerpentInventory.mat`

  Burnup-dependent isotopic mass densities extracted from Serpent2 for all isotopes tracked by NMB.

* `<library>_<REACTOR>_<enrichment>_SerpentInventory_MDENS.xlsx`

  Excel version of the isotopic inventory data.

## Support Modules

### `support_modules/`

Contains auxiliary MATLAB functions and supporting nuclear data files used during microscopic cross-section extraction and data processing.

The directory includes:

* decay matrices derived from ENDF/B-VIII.1, JEFF-4.0, and JENDL-5.0;
* isotope mapping utilities;
* NMB isotope ordering definitions;
* ORIGEN-based neutron yield data;
* supporting functions required by the extraction workflow.

Further details are provided in the README file contained within the `support_modules/` directory.

---

## Main MATLAB Scripts

### `microXS.m`

Main processing script.

Traverses the `data/` directory structure, extracts microscopic cross sections from all available depletion matrices, and generates NMB-formatted cross-section libraries.

### `extract_microXS.m`

Extracts one-group microscopic cross sections from a single Serpent2 depletion matrix.

### `extract_SerpentBU.m`

Extracts isotopic mass densities from Serpent2 depletion results for all isotopes tracked by NMB as a function of burnup and irradiation time.

### `ZAI2Name.m`

Converts ZAI identifiers into standard isotope names.
